Creating new icons We have many icons for you to choose from, but there may be times when these don’t clearly represent a particular concept. If this is the case, you can follow these guidelines to create new icons. Solid shapes Creating a solid-shape icon should always be favoured over creating an outline icon. This might be difficult in certain cases, such as the search icon. Front view A flat front view should be the default choice. Anything else, such as isometric perspective, should only be used if it’s absolutely necessary to clearly represent the concept. Use square shapes You should make sure the proportions are as close to a square as possible. Very rectangular shapes should be tilted to match this criteria. To do this you need to imagine that the icon is placed in a square container. Use round corners Rounded shapes and corners should always be favoured over harsh pointed shapes. Need more help?
