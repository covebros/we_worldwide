Print Kievit Our primary typeface for print material is Kievit. We’ve chosen this typeface because of its simple, clean lines and elegant shapes. Arial Our secondary typeface is Arial, it can be used for internal documents and when Kievit isn’t available. For more guides on using typography for print, please visit Creative Bank. Digital Open Sans Open Sans is the primary typeface used across all digital channels. We’ve chosen Open Sans because of its neutral and friendly appearance. It’s suitable for web and mobile interfaces and it’s easy to read. You should use Open Sans for titles, sub-headings and body copy highlights. Arial Arial is our secondary digital typeface. It's used for body copy, paragraphs, tables and forms - especially when the content requires maximum compatibility between the browser and operating systems.

Arial should be used when Open Sans isn’t available. For more guides on using typography for digital, please visit Creative Bank. Using fonts other than the ones described in this guide can lead to licence infringement or rendering problems in web browsers. If in doubt, contact the Brand Team.
